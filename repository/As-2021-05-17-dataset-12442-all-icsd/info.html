As-2021-05-17-dataset-12442-all-icsd (recommend)¶

The current structure dataset comprises 12442 structures generated from unique ICSD prototype structures composed of single elements with zero oxidation state. A more detailed procedure is found in Phys. Rev. B 99, 214108 (2019).

Improvement from 2020---dataset-10000-all-icsd

  • More robust for structures with a small interatomic distance

  • More complex potential models are included.

  • MLPs are estimated by using small regression weights for energetically unstable structures.

  • MLPs are estimated without using DFT stress tensors.

The procedure to estimate interatomic potentials from the dataset is found in Phys. Rev. B 99, 214108 (2019) and Phys. Rev. B 102, 174104 (2020).
