1. Installation¶

1.1. Required C++ libraries¶

  1. Armadillo (required version >= 8.600)

  2. boost C++ Libraries

  3. libboost_python3

  4. GNU Scientific Library (GSL)

1.2. Required Python modules¶

  1. numpy

  2. scipy

  3. scikit-learn

  4. joblib

  5. sympy (optional)

  6. spglib (optional)

  7. pymatgen (optional)

1.3. Required C++ libraries to build only a library for lammps¶

  1. Armadillo (required version >= 8.600)

  2. boost C++ Libraries

No python modules are required.

1.4. How to build libraries on pyml¶

  1. Modify links for armadillo and libboost_python3 libraries, and include path for python.h in $(pyml)/c++/makefile.

  2. Build two libraries of pymlcpp.so (used by python codes) and libmlip.so (used by C++ codes such as lammps) as

    pyml=~/git/pyml
cd $(pyml)/c++
make

    pymlcpp.so and libmlip.so are generated in $(pyml)/c++/lib. When building only libmlip.so, use “make lammps” instead of “make” as

    pyml=~/git/pyml
cd $(pyml)/c++
make lammps

    Only libmlip.so is generated in $(pyml)/c++/lib.

  3. Set LD_LIBRARY_PATH for pymlcpp and libmlip.so as

    export LD_LIBRARY_PATH=$(pyml)/c++/lib:$LD_LIBRARY_PATH

    If required, set additional LD_LIBRARY_PATH for other libraries such as boost

    export LD_LIBRARY_PATH=~/usr/local/boost/lib:$LD_LIBRARY_PATH

  4. Set PYMLPATH to use group-theoretical invariants as

    PYMLPATH=$(pyml)/c++/lib
