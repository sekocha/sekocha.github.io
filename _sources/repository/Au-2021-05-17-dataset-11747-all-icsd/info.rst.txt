Au-2021-05-17-dataset-11747-all-icsd (recommend)
====================================================

.. image:: summary/pareto.png
   :scale: 20%

The current structure dataset comprises 11747 structures generated from unique ICSD prototype structures composed of single elements with zero oxidation state. A more detailed procedure is found in `Phys. Rev. B 99, 214108 (2019) <https://journals.aps.org/prb/abstract/10.1103/PhysRevB.99.214108>`_.

**Improvement from 2020-**-**-dataset-10000-all-icsd**

* More robust for structures with a small interatomic distance
* MLPs are estimated by using small regression weights for energetically unstable structures.
* MLPs are estimated without using DFT stress tensors.

The procedure to estimate interatomic potentials from the dataset is found in `Phys. Rev. B 99, 214108 (2019) <https://journals.aps.org/prb/abstract/10.1103/PhysRevB.99.214108>`_ and `Phys. Rev. B 102, 174104 (2020) <https://journals.aps.org/prb/abstract/10.1103/PhysRevB.102.174104>`_.
